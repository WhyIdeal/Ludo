# Ludo

A Minimal And Lightweight Online Multiplayer Ludo Experience Built With Kotlin & Firebase, Designed For Couples And Friends.

## Features

- Online Multiplayer
- Fair & Random Dice Logic
- Resume Previous Match
- Dark & Light Theme
- Smooth Animations
- Lightweight & Optimized
